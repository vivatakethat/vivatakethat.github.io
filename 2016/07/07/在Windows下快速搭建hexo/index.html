hexo博文分类数和标签数显示不正确？

  • 先移除博客根目录下的.deploy_git文件夹和 db.json文件，然后清理下 public 目录下的categories 和 tags 两个文件夹里的文件，然后重新部署静态博客：

hexo clean #清除缓存
hexo g #生成静态网页
hexo s #启动服务预览
hexo d #开始部署
