管理书签
若要添加书签，请将光标置于要添加书签的行上。 单击“切换”按钮或按 CTRL+K。 这样便可添加书签。 再次单击“切换”按钮（或按 CTRL+K），将会删除该书签。 也可以单击书签窗口中的“删除”按钮来删除书签。

Ctrl + K Ctrl + K = 将当前行添加书签
　　
Ctrl + K Ctrl + N = 导航至下一个书签
